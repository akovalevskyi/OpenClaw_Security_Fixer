# OpenClaw Security Threat Model

This document outlines the potential threats to an OpenClaw AI agent environment and how the **OpenClaw Security Toolkit** mitigates them.

## 1. Threat Actors
- **External Script Kiddies:** Automated bots scanning for open ports (SSH, Control UI).
- **Targeted Prompt Injectors:** Attackers attempting to trick the agent into leaking secrets or executing commands.
- **Malicious Indirect Content:** Poisoned web pages or documents the agent reads via tools.
- **Compromised Supply Chain:** Vulnerabilities in upstream npm/pip packages or base Docker images.

## 2. Attack Vectors & Mitigations

| Attack Vector | Impact | Mitigation in Toolkit |
| :--- | :--- | :--- |
| **Control UI Bypass** | Full takeover of the agent gateway. | `GW_INSECURE_AUTH`, `GW_NO_DEVICE_AUTH` checks. |
| **Prompt Injection** | Agent ignores instructions, leaks identity, or abuses tools. | `WEAK_PROMPT_INTEGRITY` audit & Model hardening guide. |
| **Remote Code Execution (RCE)** | Execution of arbitrary shell commands on the VPS. | `SANDBOX_OFF`, `FS_NOT_ISOLATED`, `TELEGRAM_DENY_EXEC` checks. |
| **Financial DoS** | Massive API costs due to infinite loops or bot spam. | `NO_RATE_LIMITS`, `NO_AGENT_LIMITS` checks. |
| **Secret Exfiltration** | Theft of OpenAI/Groq/AWS keys from disk or environment. | `DATA_LEAK_IN_WORKSPACE`, `HARDCODED_KEY`, `DOCKER_SOCK_MOUNTED`. |
| **Container Escape** | Malicious agent breaks out of Docker to host OS. | `RUNNING_AS_ROOT`, `DOCKER_SOCK_MOUNTED`, `no-new-privileges` checks. |

## 3. Security Boundaries
1. **The Sandbox:** The primary cage for the AI's execution.
2. **The Gateway:** The gatekeeper for all incoming messages and outgoing tool calls.
3. **The Vault:** The secure bunker for secrets.
4. **The User Approval:** The final human-in-the-loop safety net.

## 4. Remaining Risks (Future Work)
- **Zero-Day LLM Vulnerabilities:** New jailbreak techniques that bypass established safety filters.
- **Advanced Social Engineering:** The agent being tricked into long-term persuasion attacks (Crescendo).
- **Sophisticated Supply Chain Attacks:** Malicious code injected into legitimate dependencies.
